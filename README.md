# BalancedBrackets

## How it works:

Given a string, return true if it contains all balanced parenthesis (), curly-brackets {}, and square-brackets [].

## Examples:

INPUT: str: "(x + y) - (4)" -------> OUTPUT: true;

INPUT: str: "(((10 ) ()) ((?)(:)))" -------> OUTPUT: true;

INPUT: str: "(50)(" -------> OUTPUT: false;

INPUT: str: "[{]}" -------> OUTPUT: false;
